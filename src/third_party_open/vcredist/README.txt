This folder contains the Microsoft Visual C++ Runtime libraries.

Currently this only includes versions:
10.0 (VS2010)
11.0 (VS2012)
12.0 (VS2013)
14.0 (VS2015)

* C Runtime Library (CRT) for native code.
* Standard C++ Library for native code.


NOTE: Debug versions are not included, developers should have them on their own machine.
